# Azure Resource Organization and Governance Report

## Project Overview

This project demonstrates how Azure resources can be organized and governed using Resource Groups, naming conventions, tagging strategies, and Role-Based Access Control (RBAC). The implementation simulates a real-world multi-environment cloud infrastructure containing Development and Production environments.
